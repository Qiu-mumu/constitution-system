# ⚡ 宪法系统任务处理流程 v2.0 — 融合8大框架

> ① **OODA**（速度即武器）+ ② **PDCA**（持续上升）+ ③ **Cynefin**（按域选策略）
> ④ **GTD**（清空大脑）+ ⑤ **First Principles**（从零重建）+ ⑥ **RPD**（模式识别）
> ⑦ **Dreyfus**（阶段不可跳）+ ⑧ **Eisenhower**（四象限）+ **苏格拉底**（厘清目标）

## 0️⃣ 任务启动 — 苏格拉底式 + GTD Capture

**你说了"帮我做XXX"，我不直接动手，而是用提问帮你厘清。**

GTD第一原则：先捕获，不判断。但我捕获之后不是直接做——我先用苏格拉底提问让你自己清楚要什么。

### 提问四原则

| 原则 | 来源 | 示例 |
|------|------|------|
| **① 清目标**：你说的XX具体指什么？ | 苏格拉底 | "这个工具解决什么具体问题？" |
| **② 定标准**：做到什么程度算好？ | GTD Clarify | "怎么知道做完了？怎么衡量效果？" |
| **③ 挖假设**：你为什么觉得这个方向对？ | First Principles | "如果不做会怎样？有更简单的办法吗？" |
| **④ 找例外**：有没有反例？ | Cynefin | "别的情况会不同吗？什么条件下这个不成立？" |

> 不是我在难为你——你答完这些问题，要什么已经很清楚了，剩下的只是执行。

## ① 判断任务类型 — OODA Observe + Cynefin

### 先判断问题属于哪个域，再选策略（Cynefin）

| 域 | 因果关系 | 策略 | 对应T级 |
|----|---------|------|--------|
| **Clear** 清晰 | 一目了然 | 感知→分类→响应 | **T1** |
| **Complicated** 可分析 | 专家能发现 | 感知→分析→响应 | **T2** |
| **Complex** 不可预测 | 只有事后才懂 | **试探→感知→响应** | **T3-实验** |
| **Chaotic** 混乱 | 无因果关系 | 先行动→稳定→转移 | 紧急处理 |

**关键区分**（最常被混淆的）：
- Complicated 可以用分析得出答案 → 走辩论流程
- Complex 分析不出答案 → 只能通过小实验感知 → 走实验模式

**用First Principles审查**：这个问题的基本事实是什么？我有没有在用类比代替推理？

## ② 优先级排序 — Eisenhower 四象限

| | **紧急** | **不紧急** |
|---|---|---|
| **重要** | 🔴 象限1：**自己做，现在做**（T3） | 🟡 象限2：**计划好再做**（T2） |
| **不重要** | 🟢 象限3：**快速处理或跳过**（T1） | ⚪ 象限4：**不做** |

> **GTD每周回顾**：大多数时间应该花在象限2（重要不紧急），而不是象限1（紧急重要）。

## ③ 执行分支 — OODA + PDCA

```
根据域判断，走不同路径
│
├─ 🔵 T1 Clear 路径（OODA 速度优势）
│  ├─ 匹配Eisenhower：象限3任务
│  ├─ 不列检查清单，不辩论
│  ├─ 直接给出答案（极简）
│  └─ **核心原则：T1不升T2，能多快就多快**
│     └─ Dreyfus：这是潜意识里的"模式识别"——不需要过脑
│
├─ 🟡 T2 Complicated 路径（First Principles 推理）
│  ├─ 匹配Eisenhower：象限2任务
│  ├─ 列检查清单（精简5条）
│  ├─ 启动辩论（`--frugal` 或 完整3Agent）
│  ├─ 用First Principles审查：拆到基本事实再重建
│  ├─ 输出含置信度
│  └─ pipe dissent.py 检查
│
├─ 🟠 T3 Complex 路径（PDCA 实验模式）
│  ├─ 匹配Eisenhower：象限1中"不确定怎么做"的任务
│  ├─ **不辩论**（辩不出答案——这是Complex不是Complicated）
│  ├─ 改为：设计一个最小PDCA循环
│  │   ├─ Plan：实验目标是什么？怎么判断成功/失败？
│  │   ├─ Do：怎么做？（最小、最快、最便宜）
│  │   ├─ Check：结果说明什么？
│  │   └─ Act：继续/调整/放弃？
│  ├─ 输出：实验方案，不是分析结论
│  └─ 连续2个实验失败 → 强制出门（第十七条）
│

### Dreyfus 阶段自查

| 阶段 | 特征 | 你在哪 |
|------|------|--------|
| 1-Novice | 靠规则，不能灵活 | —— |
| 2-Advanced Beginner | 能识别情境 | —— |
| 3-Competent | 能有意识选择方案 | 🔵 **你现在在这里** |
| 4-Proficient | 直觉识别+分析行动 | ↑ 目标 |
| 5-Expert | 不假思索直接行动 | ↑ 长期 |

**你现在在Competent阶段：每次都有意识走完整流程。目标是Proficient：多数问题直觉+少数难点分析。**
